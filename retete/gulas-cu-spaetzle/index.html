Fel secundar

Gulas cu Spaetzle

Se curata ceapa, cartofii si usturoiul.

Înapoi la categorie

Detalii rețetă

CategorieFel secundar
EchipamentLa oală

Înainte să începi

O verificare rapidă ca să gătești mai liniștit.

Ingrediente

  • Porc (500g)
  • Ceapa mare (2 buc.)
  • Usturoi (2 catei)
  • Spaetzle (~250g)
  • Cartofi (~500g)
  • Mix legume (ceap, morcov, ardei, patrunjel radacina, pastarnac radacina)
  • Apa
  • Sare*
  • Boia*
  • Piper negru proaspat rasnit*
  • Ulei floarea soarelui
  • Pasta gulas*
  • Pasta tomate*
  • Frunze patrunjel*
  • *dupa gust

Mod de preparare

  1. Se curata ceapa, cartofii si usturoiul.
  2. Cartofii se taie colturi, se spala si se lasa in apa.
  3. Se toaca ceapa marunt.
  4. Se taie carnea bucati (cuburi).
  5. Intr-o oala se pune ulei si se lasa la incalzit pe foc mediu. Dupa ce se incalzeste se adauga ceapa si legumele si se calesc timp de aproximativ 3-5 minute. Se adauga carnea si se rumeste impreuna cu mixul de legume si ceapa timp de alte 5-10 minute. La compozitie se adauga si usturioul zdrobit si se lasa la calit inca ~1 minut.
  6. Se adauga apa cat sa acopere compozitia din oala, impreuna cu sare, boia, piper negru, pasta gulas si pasta tomate si se lasa la fiert pentru ~5 minute.
  7. Se adauga cartofii si apa suficienta pentru a acoperi intregul continut al oalei.
  8. Se lasa la fiert ~30 minute timp in care, in alta oala se fierb spaetzle (in ultimele ~10 minute). Se adauga spetzle in oala si se fierbe amestecand continu pana scade apa
  9. Se taie frunzele de patrunjel si se adauga peste mancarea fara a amesteca.

Poftă bună!
