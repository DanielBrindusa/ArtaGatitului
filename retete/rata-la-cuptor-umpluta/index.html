Fel secundar

Rata la cuptor umpluta

Se spala bine rata si se usuca cu prosoape de hartie. Se condimenteaza uniform cu sare, piper si boia si se lasa la marinat ~24h.

Înapoi la categorie

Detalii rețetă

CategorieFel secundar
EchipamentNecesită cuptor, Necesită tavă

Înainte să începi

O verificare rapidă ca să gătești mai liniștit.

Ingrediente

  • Rata:
  • Rata (~2kg)
  • Unt (250g)
  • Apa (250ml)
  • Sare*
  • Piper*
  • Boia*
  • Umplutura:
  • Ceapa marime medie (1 buc.)
  • Usturoi (2 catei)
  • Merisoare (200g)
  • Portocala/Mar/Para/Gutuie (2 buc.)
  • Orez (200g)
  • Ulei
  • Garnitura:
  • Cartofi (1,5kg)
  • Portocala/Mar/Para/Gutuie (1 buc.)
  • Sos:
  • Zeama lamaie (1/2 buc.)
  • Zeama portocale (2 buc.)
  • Zahar (200g)
  • Unt (40g)
  • Coniac/Esenta rom (50ml)
  • Scortisoara*
  • Cuisoare*
  • *dupa gust

Mod de preparare

  1. Marinat:
  2. Se spala bine rata si se usuca cu prosoape de hartie. Se condimenteaza uniform cu sare, piper si boia si se lasa la marinat ~24h.
  3. Umplutura si garnitura:
  4. Se curata ceapa, usturoiul, cartofii si fructele.
  5. Se toaca marunt ceapa, se taie fructele si cartofii colturi.
  6. Se spala orezul bine.
  7. Se pune ulei intr-o tigaie medie si se calesc ceapa, merisoarele si orezul pentru 3-5 minute. Se adauga fructele, usturoiul zdrobit, sare, piper si boia si se calesc pentru alte ~2 minute.
  8. Compozitia rezultata se baga in rata, se inchide bine rata, se unge cu unt uniform, se cresteaza pieptul ratei si se adauga felii de portocale in crestaturi.
  9. Rata se aseaza intr-o tava medie impreuna cu apa, se acopera cu folie de aluminiu si se lasa la cuptor ~1,5 ore la aproximativ 180°C.
  10. Se inlatura folia de aluminiu, se unge rata cu lichidul din tava si se adauga fructele si cartofii in tava pentru alte ~40 minute.
  11. Rata se unge cu lichidul din tava frecvent.
  12. Sosul:
  13. Se curata fructele de coaja, se taie colturi si zeama unei jumatati de lamaie se adauga peste ele.
  14. Intr-o oala mica se caramelizeaza zaharul la foc mic. Odata ce zaharul a fost complet caramelizat se adauga untul si zeama a doua portocale si se lasa la foc mic timp de ~5 minute.
  15. Se adauga scortisoara fructele, coniac/esenta rom si cuisoarele. Se lasa la fiert la foc mic timp de ~5 minute. Dupa oprirea focului se scot cuisoarele din sos.

Poftă bună!
