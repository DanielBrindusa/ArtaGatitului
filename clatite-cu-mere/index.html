Mic dejun

Clatite cu mere

Marul se spala, se curata de coaja, se taie in sferturi, se indeparteaza cotorul, iar pulpa fructului se da pe razatoare fina.

Înapoi la categorie

Detalii rețetă

CategorieMic dejun
EchipamentLa tigaie

Înainte să începi

O verificare rapidă ca să gătești mai liniștit.

Ingrediente

  • Mar (1 buc.)
  • Faina (180g)
  • Praf de copt (1 ½ lingurite)
  • Bicarbonat de sodiu (½ lingurita)
  • Scortisoara (1 lingurita)
  • Lapte (240ml)
  • Sirop artar (4 linguri)
  • Sirop de mere (1 lingurita)
  • Sare*
  • Nutella
  • Fructe
  • *dupa gust

Mod de preparare

  1. Marul se spala, se curata de coaja, se taie in sferturi, se indeparteaza cotorul, iar pulpa fructului se da pe razatoare fina.
  2. Faina se amesteca cu praful de copt, bicarbonat de sodiu, scortisoara si sare. Laptele se amesteca cu siropul de artar si otetul de mere. Ingredientele se amestega intr-un aluat omogen.
  3. Se incalzeste tigaia de clatite si se toarna forme micute de aluat de clatite cu polonicul.
  4. Se coc ~3 minute pe fiecare parte.
  5. Se taie fructele si se servesc impreuna cu nutella si clatitele calde.

Poftă bună!
