Adaugă rețetă

Această pagină te ajută să creezi o rețetă fără să scrii cod. Previzualizarea se actualizează automat, iar exportul rămâne compatibil cu site-ul static de pe GitHub Pages.

Cum adaugi o rețetă nouă

Această pagină te ajută să creezi vizual o rețetă și să exportezi datele potrivite pentru proiect. Ea nu publică singură rețeta online, deoarece site-ul este static și rulează pe GitHub Pages.

1. Completează rețeta

  1. Scrie titlul rețetei.
  2. Alege categoria exactă din listă.
  3. Adaugă descrierea scurtă.
  4. Adaugă ingredientele și pașii de preparare.
  5. Verifică previzualizarea din dreapta.

2. Ce înseamnă slug / link

Slug-ul este partea din link a rețetei. Se generează automat din titlu, fără diacritice, fără spații și cu litere mici.

Exemplu: „Supă de pui cu găluște” devine supa-de-pui-cu-galuste, iar pagina generată va fi retete/supa-de-pui-cu-galuste/index.html.

3. Ce copiezi în proiect

Copiază obiectul exportat și lipește-l în LOCAL_FALLBACK_RECIPES din build-static-site.mjs, la finalul listei.

Păstrează virgula dintre obiecte. Nu edita direct assets/js/recipes.js ca sursă principală, fiindcă generatorul îl rescrie.

4. Publicare pe GitHub Pages

  1. Salvează modificarea în build-static-site.mjs.
  2. Rulează node build-static-site.mjs pe calculatorul tău.
  3. Urcă pe GitHub fișierele schimbate, inclusiv assets/js/recipes.js, retete/<slug>/index.html și <slug>/index.html.
  4. Așteaptă redeploy-ul GitHub Pages, apoi testează rețeta pe site.

5. Greșeli frecvente

  • Lipsește virgula dintre rețete în LOCAL_FALLBACK_RECIPES.
  • Slug-ul este duplicat.
  • Categoria nu este una dintre categoriile existente.
  • Au rămas rânduri goale la ingrediente sau pași.
  • Ai modificat fișierul, dar nu ai rulat generatorul.
  • GitHub Pages încă nu a terminat publicarea.

6. Verificare finală

  • Rețeta apare în căutare.
  • Rețeta apare în categoria corectă.
  • Pagina rețetei se deschide corect.
  • Cardul arată bine pe mobil și desktop.
  • Randomizer-ul o poate folosi dacă categoria este inclusă.
Notă despre căutare și „Ce pot găti?”: potrivirile funcționează pe cuvinte complete. Pentru rezultate mai bune, folosește ingrediente și cuvinte-cheie simple, de exemplu ou, ouă, pui, cartofi.
