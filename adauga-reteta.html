Adaugă rețetă

Această pagină te ajută să creezi o rețetă fără să scrii cod. Previzualizarea se actualizează automat, iar exportul rămâne compatibil cu site-ul static de pe GitHub Pages.

Cum funcționează

  1. Completează titlul și categoria.
  2. Adaugă ingredientele.
  3. Adaugă pașii de preparare.
  4. Verifică previzualizarea.
  5. Exportă rețeta.
  6. Adaugă blocul exportat în proiectul GitHub și rulează generatorul.

Pentru că site-ul este static, salvarea directă în site nu este posibilă fără conectare la GitHub. Ciornele salvate local rămân doar în acest browser.
