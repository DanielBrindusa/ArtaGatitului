Salata de ciuperci

Se curata, se spala ciupercile si se taie marunt.

Înapoi la categorie

Detalii rețetă

CategorieSalate
EchipamentLa tigaie

Înainte să începi

O verificare rapidă ca să gătești mai liniștit.

Ingrediente

  • Ciuperci champignon (250g)
  • Usturoi (2 catei)
  • Cum maggi (1 buc.)
  • Faina (1 lingura)
  • Parmezan (100g)
  • Maioneza (2 linguri)
  • Iaurt (2 linguri)
  • Apa (100ml)
  • Ulei floarea soarelui
  • Paine
  • Rosii cherry
  • *dupa gust

Mod de preparare

  1. Pregatire:
  2. Se curata, se spala ciupercile si se taie marunt.
  3. Se rade parmezanul.
  4. Intr-o tigaie medie se pun ciupercile la calit cu usturoiul zdrobit si putin ulei pe foc mediu timp de ~5 minute. Se adauga un cub maggi, apa si faina si se lasa la fiert pana se omogenizeaza, dupa care se adauga si parmezanul.
  5. Compozitia se lasa la racit ~30 minute. Cand este rece se adauga maioneza si iaurtul si se amesteca bine.
  6. Se serveste pe paine prajita si cu rosii cherry.

Poftă bună!
